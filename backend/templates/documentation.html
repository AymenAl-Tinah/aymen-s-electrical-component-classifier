Project Overview

The Electrical Component Classifier is a state-of-the-art web application that leverages deep learning and computer vision to automatically identify and classify electrical components from images. This professional-grade system combines the power of artificial intelligence with an intuitive user interface to provide accurate, real-time component classification.

Technology Stack

Deep Learning

PyTorch 2.1.0 with ResNet50 architecture for high-accuracy image classification

Backend

Flask 2.3.3 framework providing robust REST API endpoints and server logic

Frontend

Modern HTML5, CSS3, and JavaScript with responsive design and smooth animations

Computer Vision

Advanced image preprocessing with torchvision and PIL for optimal model performance

Key Features

  • 18 Component Types: Accurately classifies capacitors, resistors, diodes, ICs, LEDs, transformers, and more
  • High Accuracy: 73%+ classification accuracy on test datasets
  • Real-time Processing: Fast inference with results in under 2 seconds
  • Drag & Drop Upload: Intuitive file upload with visual feedback
  • Top 5 Predictions: View confidence scores for multiple predictions
  • Mobile Responsive: Works seamlessly on desktop, tablet, and mobile devices
  • Professional UI: Modern design with smooth animations and transitions
  • Export Results: Download classification results as JSON format

Model Architecture

The classifier uses a ResNet50 convolutional neural network as the backbone, pre-trained on ImageNet and fine-tuned on the ElectroCom61 dataset. The architecture includes:

  • Backbone: ResNet50 (frozen layers for transfer learning)
  • Custom Classifier: 3-layer fully connected network (2048 → 512 → 18 classes)
  • Input Size: 128×256 pixels with ImageNet normalization
  • Activation: ReLU activation with Dropout (0.5) for regularization
  • Output: Softmax probabilities for 18 electrical component classes

How to Use

  1. Upload Image: Drag and drop or click to select an electrical component image
  2. Preview: Review the uploaded image before classification
  3. Classify: Click the "Classify Component" button to run AI prediction
  4. View Results: See the predicted component type with confidence score
  5. Download: Optionally export the results as JSON for further analysis
